# Showcase examples

Fictional-but-plausible mock-up projects that demonstrate the
universal-spawn standard working across the ecosystem. None of these
ship as code — each one is a README documenting the manifest the
project uses, why it picked the platforms it picked, and how
discovery happens.

| File | The project it pretends to be |
|---|---|
| [`x-reply-bot.md`](./x-reply-bot.md) | An X reply bot powered by Grok |
| [`notion-template-pack.md`](./notion-template-pack.md) | A Notion template pack |
| [`figma-design-system.md`](./figma-design-system.md) | A Figma design system |
| [`hf-space-classifier.md`](./hf-space-classifier.md) | A Hugging Face Space |
| [`chrome-receipts.md`](./chrome-receipts.md) | A Chrome extension |
| [`unity-asset-pack.md`](./unity-asset-pack.md) | A Unity asset pack |
| [`godot-roguelike.md`](./godot-roguelike.md) | A Godot game |
| [`vscode-parchment-theme.md`](./vscode-parchment-theme.md) | A VS Code theme extension |
| [`modal-data-pipeline.md`](./modal-data-pipeline.md) | A data pipeline on Modal |
| [`discord-mod-bot.md`](./discord-mod-bot.md) | A Discord bot |
| [`arduino-weather-station.md`](./arduino-weather-station.md) | An Arduino weather station |
| [`mcp-github-server.md`](./mcp-github-server.md) | An MCP server |
| [`crewai-research.md`](./crewai-research.md) | A research agent using CrewAI |
| [`nextjs-saas-starter.md`](./nextjs-saas-starter.md) | A NextJS SaaS starter |
| [`roblox-museum.md`](./roblox-museum.md) | A Roblox experience |

Each example is a one-stop story of what the manifest enables.
