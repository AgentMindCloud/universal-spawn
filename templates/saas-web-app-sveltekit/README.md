# `saas-web-app-sveltekit` template

SvelteKit SaaS starter manifest. Targets Cloudflare Pages by default
(SvelteKit's `@sveltejs/adapter-cloudflare`).

## Change before shipping

1. `metadata.author` and `metadata.source.url`.
2. If you swap to Vercel, replace the `platforms.cloudflare` block
   with `platforms.vercel` and use `@sveltejs/adapter-vercel`.
